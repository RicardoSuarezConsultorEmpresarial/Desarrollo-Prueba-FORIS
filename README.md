# Desarrollo-Prueba-FORIS

Desarrollo Prueba FORIS

Este ejercicio se trabajó primero desde Python para validar que la información de ambas bases estuviera alineada antes de mover la lógica al motor final.
La primera fase consiste en revisar estructura, tipos, nulos, duplicados y coincidencia entre registros para asegurar que la consolidación sea confiable.
Una vez validada la data, se genera un archivo intermedio en CSV para facilitar la revisión en Excel y, más adelante, la carga o el cruce dentro del motor destino.
